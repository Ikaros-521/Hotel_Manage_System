# Hotel_Manage_System
# 酒店管理系统

#### 介绍
C++酒店管理系统，已经更新Windows版本，内置readme说明，可自行使用。

#### 使用说明

1. 终端执行 make
2. 终端执行 ./hotel
3. 完事了

#### 相关说明

1. 数据保存在data中，程序读写文件，所以如果需要调试可以直接改数据
